Welcome to the Ontology Browser

This application allows you to browse and search an ontology knowledge graph.

User Guide

Here's how you can use the application:

  • Search Dataset: Enter a SPARQL query in the search field to retrieve specific data from the ontology.
  • Display Data: View all classes and instances defined in the ontology.
  • SPARQL Queries: Run predefined SPARQL queries to explore common data patterns.
  • Edit Knowledge Graph: Modify the ontology by adding or updating entries.
